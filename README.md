# 🌊 Smart Disaster Response & Flood Prediction

## 📌 Overview

Smart Disaster Response & Flood Prediction is an AI-powered Decision Intelligence Platform developed for the *Gen AI Academy Hackathon*. The application helps citizens and authorities make informed decisions during disasters by combining AI, weather information, and disaster reports.

---

## 🚀 Features

- 🤖 Gemini AI disaster analysis
- 🌦️ Live weather information
- 📤 Disaster report submission
- 🖼️ Image upload
- 📊 Analytics dashboard
- 💾 SQLite database
- 🏠 Nearby emergency shelter suggestions
- 📋 Disaster report history
- 📱 Responsive web interface

## ⚙️ Installation

bash

bash

Run the application:

bash
python app.py


Open:


http://127.0.0.1:5000
